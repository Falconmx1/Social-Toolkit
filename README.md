# 🛡️ Social Toolkit - Ethical Awareness Edition

**Herramienta educativa para Linux** diseñada para demostrar y prevenir ataques de ingeniería social en entornos controlados con autorización previa.

> ⚠️ **Uso exclusivo para profesionales de seguridad, administradores de sistemas y estudiantes.**  
> El mal uso de esta herramienta en sistemas sin autorización explícita es ilegal. El autor no se hace responsable.

## 📦 Características (modo educativo)
- Generación de páginas de phishing clonadas (solo demostración)
- Simulación de ataques spear-phishing en entornos aislados
- Detección de credenciales débiles mediante listas comunes
- Auditoría de políticas de seguridad humana
- Reportes educativos sobre cómo mitigar riesgos

## 🖥️ Requisitos
- Linux (Kali, Ubuntu, Debian)
- Python 3.8+
- Bash
- Dependencias: `pip install -r requirements.txt`
